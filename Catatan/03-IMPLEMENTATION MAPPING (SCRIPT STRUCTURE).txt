🧠 2. CORE MODULE MAPPING (v1.1 → SCRIPT)
📊 2.1 STAT SYSTEM → StatSystem Module
🎯 Tugas:

Mengelola:

base stats
allocation points
caps
reset system
📦 Functions:
GetBaseStats(player)
ApplyStatPoint(player, statType, amount)
ResetStats(player)
GetDerivedStats(player)
ValidateStatCap(player)
⚔️ 2.2 COMBAT SYSTEM → CombatSystem Module
🎯 Tugas:

Orchestrator semua combat flow

Urutan v1.1 kamu:

Hit → Defence → Crit → Damage
📦 Functions:
ProcessAttack(attacker, target)
ResolveHit(attacker, target)
ResolveCrit(attacker)
CalculateFinalDamage(attacker, target)
ApplyDamage(target, damage)
🎲 2.3 HIT SYSTEM → HitSystem Module
🎯 Logic dari v1.1:
Final Hit Chance = Hit Rating - Dodge
Clamp 0–100
📦 Functions:
CalculateHitChance(attacker, target)
RollHit(chance)
GetHitRating(player)
GetDodgeChance(player)
💥 2.4 DAMAGE ENGINE → DamageCalculator Module
🎯 Logic:
Damage = ATK × 5
Final = Damage - Defence
Min 1
📦 Functions:
GetBaseDamage(player)
ApplyDefenseReduction(damage, target)
CalculateMinimumDamage(damage)
🛡️ 2.5 DEFENSE SYSTEM → DefenseSystem Module
🎯 Logic:
Effective Defence = DEF + Armor - Penetration
📦 Functions:
GetDefense(player)
ApplyArmorPenetration(defense, pen)
CalculateEffectiveDefense(target)
💢 2.6 CRIT SYSTEM → CritSystem Module
🎯 Logic:
Crit Chance = LUCK × 0.15%
Crit Damage = Equipment Only
📦 Functions:
CheckCrit(player)
GetCritChance(player)
ApplyCritMultiplier(damage, player)
🧍 2.7 EQUIPMENT SYSTEM → EquipmentSystem Module
🎯 Tugas:
equip / unequip item
apply stat bonus
store passive effects
📦 Functions:
EquipItem(player, item)
UnequipItem(player, slot)
GetEquipmentStats(player)
GetPassiveEffects(player)
🔁 2.8 FORMULA ENGINE → FormulaEngine Module
🎯 Tugas:

Single source semua formula v1.1

📦 Functions:
CalculateHP(level, vit)
CalculateAttackSpeed(agi)
CalculateDodge(agi)
CalculateHitRating(dex)
CalculateCritChance(luck)

👉 ini “jantung matematik sistem kamu”

🎮 3. SERVER FLOW (PENTING BANGET)
⚔️ Combat Flow di Server:
Player clicks attack
↓
CombatController receives request
↓
HitSystem → check hit/miss
↓
DefenseSystem → calculate mitigation
↓
CritSystem → check crit
↓
DamageCalculator → final damage
↓
ApplyDamage to target
↓
Send result to client
